# Roadmap: iMBA Data Analytics in Marketing Study App

## Overview

A modern visual learning platform for iMBA Data Analytics in Marketing (MBA 564A). Features 4 complete course modules with interactive visualizations, animated explanations, exam preparation, and glossary tools. Deployed to Firebase Hosting.
