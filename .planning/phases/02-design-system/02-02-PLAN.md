```svelte
<script lang="ts">

  let {
    size = 'default',
    class: className = '',
    children
  }: {
    size?: 'sm' | 'default' | 'lg' | 'full';
    class?: string;
    children: Snippet;
  } = $props();

  const sizes = {
    sm: 'max-w-2xl',
    default: 'max-w-4xl',
    lg: 'max-w-6xl',
    full: 'max-w-full'
  };
</script>

<div class="mx-auto px-4 sm:px-6 lg:px-8 {sizes[size]} {className}">
  {@render children()}
</div>
```

- default: 896px max - for lecture content
- lg: 1152px max - for diagrams and visualizations
- full: no max-width - for full-bleed sections

Create a flexible grid component for layouts:

```svelte
<script lang="ts">
  import type { Snippet } from 'svelte';

  let {
    cols = 1,
    gap = 'default',
    class: className = '',
    children
  }: {
    cols?: 1 | 2 | 3 | 4 | 'auto';
    gap?: 'none' | 'sm' | 'default' | 'lg';
    class?: string;
    children: Snippet;
  } = $props();

  const colClasses = {
    1: 'grid-cols-1',
    2: 'grid-cols-1 md:grid-cols-2',
    3: 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3',
    4: 'grid-cols-1 sm:grid-cols-2 lg:grid-cols-4',
    auto: 'grid-cols-[repeat(auto-fit,minmax(280px,1fr))]'
  };

  const gapClasses = {
    none: 'gap-0',
    sm: 'gap-2',
    default: 'gap-4 md:gap-6',
    lg: 'gap-6 md:gap-8'
  };
</script>

<div class="grid {colClasses[cols]} {gapClasses[gap]} {className}">
  {@render children()}
</div>
```

- cols=2: Side-by-side comparisons (treatment vs control)
