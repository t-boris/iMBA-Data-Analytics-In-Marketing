# Plan 08-01: Glossary Data Structure & Content

## Objective

Create the data structure for glossary concepts and populate with all Module 1 terms (~30-40 concepts).

## Dependencies

- Phase 6 complete (lecture content exists to reference)
- Existing DiagramData types

## Implementation Steps

### Step 1: Create Glossary Types

Create `src/lib/glossary/types.ts`:
```typescript
interface GlossaryConcept {
  id: string;
  term: string;
  definition: string;           // 1-2 sentence definition
  extendedDefinition?: string;  // Longer explanation (markdown)
  category: ConceptCategory;
  relatedConcepts: string[];    // IDs of related concepts
  lectureId: number;            // Source lecture (1-8)
  moduleId: string;             // 'module1'
  diagramData?: DiagramData;    // Optional visualization
  tags: string[];               // For filtering
}

type ConceptCategory = 'foundation' | 'method' | 'assumption' | 'estimator' | 'bias' | 'design';
```

### Step 2: Create Module 1 Glossary Content

Create `src/lib/glossary/concepts/module1.ts`:

Key concepts to include (~35 terms):

**Foundations:**
- Causal inference, Counterfactual, Potential outcomes, Treatment effect

**Estimators:**
- ATE, ATT, ATU, LATE, ITT

**Biases:**
- Selection bias, Confounding, Omitted variable bias, Collider bias

**Methods:**
- RCT, Matching, Propensity score matching, Instrumental variables, Difference-in-differences

**Assumptions:**
- SUTVA, Unconfoundedness, Parallel trends, Exclusion restriction, Relevance condition

**Design Elements:**
- Treatment group, Control group, Randomization, Balance, Common support

**DAG Concepts:**
- DAG, Backdoor path, Collider, Mediator, Confounder node

### Step 3: Create Glossary Utilities

Create `src/lib/glossary/utils.ts`:
- `searchConcepts(query, concepts)` - fuzzy search by term/definition
- `filterByCategory(category, concepts)`
- `filterByLecture(lectureId, concepts)`
- `getRelatedConcepts(conceptId, concepts)`
- `sortAlphabetically(concepts)`
- `groupByFirstLetter(concepts)` - for A-Z index
